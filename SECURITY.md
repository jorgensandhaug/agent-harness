# Security Policy

## Supported Versions

Security fixes are applied to the latest commit on the default branch.

## Reporting a Vulnerability

Please report vulnerabilities privately and do not open public issues.

Send details to:
- Email: security@agent-harness.dev
- Subject: `[agent-harness] security report`

Please include:
- Affected version or commit SHA
- Reproduction steps and impact
- Any proof-of-concept details

We will acknowledge receipt within 3 business days and provide a remediation timeline after triage.
